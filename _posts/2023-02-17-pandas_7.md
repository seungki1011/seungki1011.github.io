## Groupby 1
### Groupby

* split -> apply -> combine의 과정을 거쳐서 연산

#### groupby의 사용

# data from: 
ipl_data = {'Team': ['Riders', 'Riders', 'Devils', 'Devils', 'Kings',

```python
df.groupby("Team")["Points"].sum() 
# "Team"은 묶음의 기준이 되는 column
# "Points"는 적용 받는 column
# sum()은 적용하는 연산
# 결과는 Team을 기준으로 Points를 sum 한 것
```
